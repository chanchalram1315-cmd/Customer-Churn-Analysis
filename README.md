# 📊 Customer Churn Analysis Dashboard

## 📌 Project Overview

This project focuses on analyzing customer churn behavior using telecom data. The objective is to identify key factors influencing customer attrition and provide actionable insights using data visualization.

---

## 🛠️ Tools & Technologies Used

* Microsoft Excel (Data Cleaning & Preparation)
* SQL (Data Analysis)
* Power BI (Dashboard & Visualization)
* DAX (KPI Calculations)

---

## 📊 Key Features

* Interactive Power BI Dashboard
* KPI Metrics:

  * Total Customers
  * Churn Customers
  * Churn Rate (%)
* Visual Analysis:

  * Churn by Gender
  * Churn by Contract Type
  * Churn by Monthly Charges
* Dynamic Filters (Slicers)

---

## 📈 Key Insights

* Customers with **month-to-month contracts** have the highest churn rate.
* **Higher monthly charges** lead to increased churn probability.
* **Long-term contracts** (1–2 years) show better customer retention.
* Gender has minimal impact on churn behavior.
